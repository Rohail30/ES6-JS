Ternary-Operator

Ternary Operator:
The ternary operator is a simplified conditional operator, providing a concise alternative to the traditional if/else statement.
Syntax: condition ? <expression if true> : <expression if false>

Example using if/else:
if (condition) {
  // expression if true;
} else {
  // expression if false;
}

Before:
if (authenticated) {
  renderApp();
} else {
  renderLogin();
}

With Ternary Operator:
authenticated ? renderApp() : renderLogin();
The ternary operator is a concise way to write conditional statements. In this example, it checks the value of authenticated. If true, it calls renderApp(); otherwise, it calls renderLogin().
