Classes

Car Class: The Car class is defined with a constructor that takes a parameter name and sets the property this.brand to the provided name. It has a method present() that returns a string indicating the statement "I have a " followed by the brand of the car.

Model Class (extends Car): The Model class is declared and extends the Car class using the extends keyword, indicating that it inherits from Car. It has its own constructor that takes two parameters (name and mod). The super(name) is called inside the Model constructor to invoke the constructor of the parent class (Car) and set the brand property. The model property is then set to the provided mod. The class has a method show() that calls the present() method from the parent class and appends information about the model. It returns a string representing a statement about the car, including both brand and model.

Instance Creation and Usage: An instance of the Model class is created with the brand "Ford" and model "Mustang" using const mycar = new Model("Ford", "Mustang"). The show() method is called on the mycar instance, and the resulting string is written to the document using document.write(). The overall result, when executed, will write the string "I have a Ford, it is a Mustang" to the document. This demonstrates the concept of inheritance, where the Model class inherits properties and methods from the Car class and adds its own functionality.

class Car {

   constructor(name) {

      this.brand = name;

   }

   present() {

      return 'I have a ' + this.brand;

   }

}

class Model extends Car {

   constructor(name, mod) {

      super(name);

      this.model = mod;

   }

   show() {

        return this.present() + ', it is a ' + this.model;

   }

}

const mycar = new Model("Ford", "Mustang");

document.write(mycar.show());
