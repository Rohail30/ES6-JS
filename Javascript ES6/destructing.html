Destructuring Arrays

Destructuring Arrays:
The first example demonstrates destructuring an array. In the "Before" section, individual elements are assigned to variables using indexing. In the "With destructuring" section, array destructuring syntax is used to directly assign values to variables.

Destructuring in Function Return:
The second example showcases destructuring when a function returns an array. The "Before" section manually assigns each element of the returned array to variables. In the "With destructuring" section, array destructuring is utilized to assign the elements in a concise manner.

Destructuring Objects in Function Parameters:
The third example illustrates destructuring objects within function parameters. In the "Before" section, a function takes an object as a parameter, and properties are accessed using dot notation. In the "With destructuring" section, the function parameter is directly destructured, simplifying property access.

Destructuring Arrays:
Before: const vehicles = ['mustang', 'f-150', 'expedition'];
const car = vehicles[0];
const truck = vehicles[1];
const suv = vehicles[2];
With destructuring: const vehicles = ['mustang', 'f-150', 'expedition'];
const [car, truck, suv] = vehicles;

Destructuring in Function Return:
Before: function calculate(a, b) {
  const add = a + b;
  const subtract = a - b;
  const multiply = a * b;
  const divide = a / b;

  return [add, subtract, multiply, divide];
}
const [add, subtract, multiply, divide] = calculate(4, 7);
With destructuring: function calculate(a, b) {
  const add = a + b;
  const subtract = a - b;
  const multiply = a * b;
  const divide = a / b;

  return [add, subtract, multiply, divide];
}
const [add, subtract, multiply, divide] = calculate(4, 7);

Destructuring Objects in Function Parameters:
Before: const vehicleOne = {
  brand: 'Ford',
  model: 'Mustang',
  type: 'car',
  year: 2021,
  color: 'red'
}

myVehicle(vehicleOne);

function myVehicle(vehicle) {
  const message = 'My ' + vehicle.type + ' is a ' + vehicle.color + ' ' + vehicle.brand + ' ' + vehicle.model + '.';
}
With destructuring: const vehicleOne = {
  brand: 'Ford',
  model: 'Mustang',
  type: 'car',
  year: 2021,
  color: 'red'
}

myVehicle(vehicleOne);

function myVehicle({type, color, brand, model}) {
  const message = 'My ' + type + ' is a ' + color + ' ' + brand + ' ' + model + '.';
}
