Spread-Operator

Using the Spread Operator in Arrays:
The JavaScript spread operator (...) allows us to quickly copy all or part of an existing array or object into another array or object.

const numbersOne = [1, 2, 3];
const numbersTwo = [4, 5, 6];
const numbersCombined = [...numbersOne, ...numbersTwo];

This example combines two arrays, numbersOne and numbersTwo, into a new array numbersCombined using the spread operator.

Using the Spread Operator in Array Destructuring:
Assign the first and second items from numbers to variables and put the rest in an array:

const numbers = [1, 2, 3, 4, 5, 6];
const [one, two, ...rest] = numbers;

This example uses array destructuring with the spread operator to assign the first item to one, the second item to two, and the rest of the items to rest.

Using the Spread Operator in Object Merging:
Combine these two objects:

const myVehicle = {
  brand: 'Ford',
  model: 'Mustang',
  color: 'red'
}

const updateMyVehicle = {
  type: 'car',
  year: 2021,
  color: 'yellow'
}

const myUpdatedVehicle = {...myVehicle, ...updateMyVehicle}

This example combines two objects, myVehicle and updateMyVehicle, into a new object myUpdatedVehicle using the spread operator.
