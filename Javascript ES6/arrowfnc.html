Arrow-Functions

Simple Function:
The first example defines a simple function named hello using the traditional function expression syntax. It has no parameters and returns the string "Hello World!".

Arrow Function:
The second example defines the same hello function using the arrow function syntax. It has the same behavior as the simple function but with a shorter syntax.

Arrow Function with Parameter:
The third example demonstrates an arrow function with a parameter (val). It returns a string by concatenating "Hello " with the provided parameter.

Arrow Function Without Parentheses:
The fourth example is a more concise form of the arrow function when it has only one parameter. The parentheses around the parameter are optional.

Simple Function:
hello = function() {
  return "Hello World!";
}


Arrow Function:
hello = () => {
  return "Hello World!";
}


Arrow Function with Parameter:
hello = (val) => "Hello " + val;


Arrow Function Without Parentheses:
hello = val => "Hello " + val;
